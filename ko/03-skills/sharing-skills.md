# 스킬 공유

이 문서는 만든 스킬을 팀·개인·플러그인 채널로 배포하는 세 가지 방법을 정리합니다.
"내 스킬을 팀에 공유하고 싶은데 어디에 올려야 하지?"라는 결정이 필요할 때 봅니다.
배포 채널마다 권한 범위와 업데이트 흐름이 달라지므로 목적에 맞게 고르세요.

## 프로젝트 스킬 (팀 공유)

1. `.claude/skills/`에 스킬을 생성합니다
2. git에 커밋합니다
3. 팀원들이 변경 사항을 pull하면 스킬이 즉시 사용 가능합니다

## 개인 스킬

```bash
# 개인 디렉토리에 복사
cp -r my-skill ~/.claude/skills/

# 스크립트를 실행 가능하게 만들기
chmod +x ~/.claude/skills/my-skill/scripts/*.py
```

## Plugin 배포

더 넓은 배포를 위해 plugin의 `skills/` 디렉토리에 스킬을 패키징합니다.
