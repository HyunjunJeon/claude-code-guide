# TypeScript v2 Preview

TypeScript v2 preview는 다음 세대 TypeScript SDK 표면을 다루는 문서입니다. 안정 버전이라기보다, 앞으로 옮겨 갈 방향을 보여주는 마이그레이션 대상에 가깝습니다.

## 언제 읽을까

다음 경우에 프리뷰 문서를 읽을 가치가 있습니다.

- 새 통합을 시작하면서 향후 표면을 보고 싶을 때
- 필요한 기능이 프리뷰 쪽에 먼저 들어왔을 때
- 현재 TypeScript 패키지에서 옮겨갈 계획을 세울 때

## 안전하게 쓰는 방법

- 도입 전에 정확한 패키지와 버전을 다시 확인합니다.
- 프리뷰 사용은 내부 호환 계층 뒤에 감춥니다.
- 정식 출시 전까지 이름과 동작이 바뀔 수 있다고 가정합니다.
- 폴백 없이 핵심 운영 경로를 프리뷰 가정 위에만 올리지 않습니다.

## 실전 조언

이미 안정적인 TypeScript 통합이 있다면, 프리뷰가 당장 해결해야 할 실제 문제를 풀어 주는 경우가 아니면 현재 GA 경로를 우선하는 편이 안전합니다.
