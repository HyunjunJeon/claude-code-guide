# Agent SDK

이 모듈은 Claude Code Agent SDK 문서군을 한국어로 정리하기 위한 허브입니다. 대화형 CLI 사용법이 아니라, 세션, 도구, 스트리밍, 권한, 배포 제어를 직접 다루는 SDK 빌더 관점이 중심입니다.

## Official reference

- [Agent SDK overview](https://code.claude.com/docs/ko/agent-sdk/overview)
- [Claude Code docs map](https://code.claude.com/docs/ko/claude_code_docs_map)
