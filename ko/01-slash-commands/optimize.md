# 코드 최적화

제공된 코드에서 다음 문제를 우선순위 순서대로 검토합니다:

1. **성능 병목 지점** - O(n^2) 연산, 비효율적인 루프 식별
2. **메모리 누수** - 해제되지 않은 리소스, 순환 참조 찾기
3. **알고리즘 개선** - 더 나은 알고리즘 또는 데이터 구조 제안
4. **캐싱 기회** - 반복 계산 식별
5. **동시성 문제** - 경쟁 조건 또는 스레딩 문제 찾기

응답 형식:
- 문제 심각도 (Critical/High/Medium/Low)
- 코드 내 위치
- 설명
- 코드 예제가 포함된 권장 수정사항
