# Agent SDK

This module collects the Claude Code Agent SDK reference set. It is for builders who want to create agents with sessions, tools, streaming, permissions, and deployment controls rather than just use Claude Code interactively.

## Official reference

- [Agent SDK overview](https://code.claude.com/docs/ko/agent-sdk/overview)
- [Claude Code docs map](https://code.claude.com/docs/ko/claude_code_docs_map)

## Recommended reading order

1. `overview.md`
2. `sessions.md`
3. `custom-tools.md`
4. `permissions.md`
5. `hooks.md`
6. `typescript.md` or `python.md`

- Explain how the SDK differs from the CLI product surface

- Keep TypeScript and Python entry points easy to find
- Cross-link SDK concepts back to the CLI guides where behavior overlaps
