이 페이지는 MCP 도구 호출 결과를 채팅 안에 인라인 UI 컴포넌트로 렌더링할 수 있는 MCP Apps 확장을 소개한다. MCP 서버가 단순 텍스트가 아니라 대시보드·폼·차트·다단계 워크플로우를 직접 보여 주고 싶을 때 검토할 옵션이다. 사용자 입력을 구조화해 받는 [mcp-elicitation.md](05-08-mcp-elicitation.md)와 짝지어 보면 인터랙티브한 MCP 경험을 더 풍부하게 설계할 수 있다.

MCP Apps는 MCP 도구 호출이 채팅 인터페이스에 직접 렌더링되는 대화형 UI 컴포넌트를 반환할 수 있게 하는 확장 기능입니다. 일반 텍스트 응답 대신, MCP 서버가 풍부한 대시보드, 양식, 데이터 시각화 및 다단계 워크플로우를 전달할 수 있으며 -- 모두 대화를 벗어나지 않고 인라인으로 표시됩니다.
