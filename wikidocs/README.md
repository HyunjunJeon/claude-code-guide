# Claude Code 빠르게 마스터 하기

`claude`를 처음 실행하는 단계부터 Memory, Skill, Subagent, MCP, Hooks, Plugin, 배포와 Agent SDK까지 학습 흐름대로 정리한 Claude Code 가이드입니다.

---

## 학습 경로

| 순서 | 주제 | 설명 |
|------|------|------|
| 00 | Claude Code 시작하기 | 설치 후 첫 작업까지 이어지는 큰 그림과 기본 워크플로 |
| 01 | CLI와 Slash Command | CLI, 대화형 모드, slash command, 출력 형식 |
| 02 | Memory | CLAUDE.md와 자동 Memory로 지속 컨텍스트 관리 |
| 03 | Skills | 재사용 가능한 지식과 워크플로를 skill로 패키징 |
| 04 | Subagents | 격리된 전문 agent와 Agent teams로 병렬 작업 분리 |
| 05 | MCP | 외부 도구, API, 실시간 데이터를 연결하는 표준 프로토콜 |
| 06 | Hooks | 이벤트 기반 자동화, 검증, 권한 통제 |
| 07 | Plugins 핵심 | plugin 개념, 설치, 운영, 보안, 배포 |
| 08 | Plugin 실전 예제 | DevOps, 문서화, PR 리뷰 plugin 실전 예제 |
| 09 | Checkpoints & Rewind | 자동 checkpoint, rewind, git·SDK 비교 |
| 10 | 고급 워크플로 | planning, routines, code review, 실행 모드 등 고급 흐름 |
| 11 | 플랫폼과 통합 | Remote Control, Web, Desktop, IDE, Slack, CI/CD 통합 |
| 12 | 설정·권한·보안 | 설정 계층, 권한, 보안, context window, 문제 해결 |
| 13 | Deployment Administration | 조직 배포, 공급자 선택, 네트워크, 데이터 정책 |
| 14 | Agent SDK | Claude Code agent loop를 앱과 백엔드에 임베드하는 SDK |
