# What's New

This document summarizes the official weekly Claude Code digest. It is the right place to scan for workflow-changing features without reading the full version-by-version changelog.

## How It Differs From The Changelog

- `Changelog`: exact releases, fixes, and patch-level behavior
- `What's new`: weekly digest of notable capabilities, demos, and why they matter

If you need to answer "what changed in my installed version," use the changelog. If you need to answer "what new feature should I care about," start here.

## Latest Official Digest Snapshot

The current official digest page lists these weekly summaries:

### Week 15 — April 6 to April 10, 2026

Version range: `v2.1.92` to `v2.1.101`

Main highlight:

- ultraplan entered early preview, letting you draft a plan in the cloud, comment on it in a browser, and either execute remotely or send it back local

Other highlighted additions:

- Monitor tool for streaming background events into the conversation
- `/loop` self-pacing when no interval is specified
- `/team-onboarding` for replayable setup guidance
- `/autofix-pr` for PR auto-fix from the terminal

Other highlighted additions:

- `/powerup` interactive lessons

- larger per-tool MCP result-size overrides

Version range: `v2.1.83` to `v2.1.85`
